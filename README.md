# **mruby/c on micro:bit for Tinybit Robot**

# 概要

本プロジェクトは、BBC micro:bit 上で動作する軽量Ruby「mruby/c」を用いて、Tinybitロボットを直感的かつ自由に制御するためのファームウェア、および高水準APIライブラリ群を提供するものです。

最終目標は、Smalrubyの設計思想に基づき、専門知識がない人でもScratchのように手軽にRuby言語でロボットプログラミングを楽しめる環境を構築することです。


mruby/c クロスコンパイラ: mrbc コマンドが実行できるように、mrubyのリポジトリから環境を構築する必要があります。（ここに詳細な手順を追記）

## 利用方法


## ブラウザからの利用方法 (Smalruby)

このファームウェアの最大の特徴は、PCに特別な開発環境を構築することなく、WebブラウザだけでRubyプログラミングとロボットの操作が完結することです。

ここでは、SmalrubyのWebサイトを利用して、Tinybitロボットを動かすまでの手順を解説します。

---

### Step 1: プログラムの作成とコンパイル

- **Smalrubyのサイトを開く**  
  WebブラウザでSmalrubyの公式サイトにアクセスし、画面上部にある「ルビー」タブを選択してコーディング画面に切り替えます。
  Smalruby:
  https://ceres.epi.it.matsue-ct.ac.jp/smt/
- **コードを記述する**  
  エディタ画面に、ロボットを動かすためのRubyコードを記述します。
  
  ```ruby
 
  # 例: 1秒ごとに前進と後退を繰り返す
  tinybit = Tinybit.new()
  
  loop do
    puts "Forward!"
    LEDMatrix.display('A')
    tinybit.move(:forward, 150, 150)
    sleep(1)
  
    puts "Back!"
    LEDMatrix.display('B')
    tinybit.move(:back, 150, 150)
    sleep(1)
  end
  ```
- **コンパイル（マイコンへの書き出し）**  
  コードが完成したら、メニューの「ファイル」タブを選択し、「マイコンへ書き出し」ボタンをクリックします。  
  → プログラムをマイコン用に変換するための、新しい画面に遷移します。

---

### Step 2: プログラムの転送と実行

#### 設定の確認
コンパイル画面が表示されたら、以下の2点を確認してください。

- コンパイラバージョン: `mrbc 3.1.0` が選択されていること。  
- マイコン: `ESP32` が選択されていること。  
  （※micro:bitですが、通信プロトコルが同じためこの設定で動作します）

#### micro:bitと接続する
- 「接続」ボタンをクリックすると、PCに接続されているシリアルポートの選択画面が表示されます。  
- `mbed Serial Port` や `USB Serial Device` など、micro:bitに対応するポートを選択して接続します。

#### 接続の確立（リセット操作）
接続後、画面下部のログに `try to enter command mode...` という表示が続く場合があります。  
これは、ファームウェアがPCからのコマンドを待っている状態です。  
この状態になったら、以下のいずれかの方法でmicro:bitを物理的にリセットしてください。

- micro:bit基板上のリセットボタンを押す  
- Tinybitに搭載されているAボタンを5秒以上長押しする  
- PCとのUSBケーブルを一度抜き差しする  

#### 接続の確認
リセットが成功し、正しく接続が確立されると、ログに以下のように表示されます。<br>
`+OK mruby/c v3.1 RITE0300 MRBW1.2`

---
### プログラムを書き込む
接続が確認できたら、「書き込み」ボタンをクリックします。  
コンパイルされたプログラムがmicro:bitのFlashメモリに転送されます。

---

### プログラムを実行する
最後に、「実行」ボタンをクリックします。  
micro:bitが自動的に再起動し、書き込まれたプログラムが実行されます。

---

## APIリファレンス
[利用可能なクラス](doc/class.md)


**主な機能**
安定したファームウェア: PCからのコマンドによるプログラム書き込みと、スタンドアローンでの自動実行に対応。

高水準Ruby API: 以下のクラス/モジュールを通じて、ロボットの機能を直感的に操作できます。

GPIO: LEDなど、汎用的な入出力を制御します。

PWM: モーターの速度など、より細かい出力を制御します。

Tinybit: モーター、RGB LED、各種センサーを統括する、ロボット専用の司令塔です。

LEDMatrix: ボード上の5x5 LEDを、1枚の電光掲示板のように扱えます。


## 動作環境
必要なもの
BBC micro:bit v2

Tinybit ロボット

USBケーブル

ソフトウェア
Arduino IDE: バージョン 1.8.19 で動作確認済み。

micro:bit ボード定義: Arduino IDEのボードマネージャから Arduino nRF528x Boards (Mbed OS) をインストールします。
